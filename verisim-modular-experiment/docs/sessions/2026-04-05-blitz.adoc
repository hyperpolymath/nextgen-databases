== Key findings

=== Path B is live — a non-trivial Core exists

After reading `arcvix-octad-data-model.tex` (formal data model) and
`rust-core/verisim-octad/src/query_octad.rs`, classification resolved:

* **Core** = `{Semantic, Temporal, Provenance}` (size 3)
* **Federable** = `{Vector, Tensor, Document, Spatial}` (size 4)
* **Conditional** = `{Graph}` (required iff cross-entity consonance claims in scope)

=== New distinction: per-octad vs store-level absence

The formal model already permits per-octad `φ(m) = ⊥` unconditionally.
**Store-level absence** is the novel axis the experiment opens — whether
a store can omit a modality entirely, not just have it be `⊥` per entity.

Temporal and Provenance are store-level-required because the enrichment
primitive (def:enrichment) writes to both on every mutation, and Identity
Persistence (inv:persist) depends on Temporal.

=== Path B vs existing architecture

Two orthogonal axes identified:

* *Location axis* (existing): standalone / federated / hybrid — where does each shape live?
* *Presence axis* (this experiment): which shapes exist at all in a federation?

These compose. A federation can be "hybrid on location axis, reduced on presence axis."

=== 5 federation-contract clauses identified

. Drift-signal projection surface
