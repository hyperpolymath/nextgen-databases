= QuandleDB Roadmap
:author: Jonathan D.A. Jewell
:email: j.d.a.jewell@open.ac.uk
:revdate: 2026-02-13

== Current Status (v0.1.0)

* [x] Julia HTTP server wrapping Skein.jl
* [x] JSON API: list, detail, statistics endpoints
* [x] ReScript frontend: dashboard, knot list, knot detail
* [x] Filtering by crossing number, genus, name
* [x] Jones polynomial display
* [x] Gauss code display
* [x] Semantic descriptor sidecar index (`quandle_semantic_index`)
* [x] Semantic equivalence candidate endpoint (`/api/semantic-equivalents/:name`)

== v0.2.0 - Query Language (KQL)

* [ ] KQL (Knot Query Language) parser
* [ ] Query DSL for topological predicates
* [ ] Query editor in frontend with syntax highlighting
* [ ] Composable predicates via text interface

== v0.3.0 - Extended Invariants

* [ ] Alexander polynomial computation and display
* [ ] HOMFLY-PT polynomial
* [ ] Knot signature
* [ ] Knot determinant

== v0.4.0 - Links and Tangles

* [ ] Multi-component link support
* [ ] Linking number computation
* [ ] Tangle storage and display

== v0.5.0 - Visualisation

* [ ] SVG knot diagram rendering
* [ ] Interactive Gauss code editor
* [ ] 3D knot viewer (optional)

== Future Directions

* Federation with other knot databases (KnotInfo, KnotAtlas)
* Batch import/export (CSV, JSON)
* User-contributed knot submissions
* WebSocket live queries
