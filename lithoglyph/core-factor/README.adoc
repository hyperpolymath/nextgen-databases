= Form.Runtime (Factor)
:toc:

The query runtime, implemented in Factor.

== Purpose

* GQL parse/plan/exec
* Query explanation (EXPLAIN)
* Schema/constraint introspection
* Provenance surfaces

== GQL PoC Subset

* `INSERT` document into collection
* `INSERT` edge (from, to, type, props)
* `SELECT` with simple predicates
* `EXPLAIN` (returns plan + reasons)
* `INTROSPECT` schema/constraints
* Optional provenance output

== Status

Implementation pending GQL grammar specification (Q-GQL-POC-001).
