= nqc — Specification Directory
:toc:

== Overview

This directory contains the canonical language/query specification files for nqc.

== Contents

=== grammar.ebnf
The canonical EBNF grammar for nqc. Covers meta-commands, query passthrough,
protocol wire format, output formatting, CLI arguments, and terminal definitions.

=== SPEC.core.scm
Formal specification in Scheme. Defines the type system, protocol invariants,
profile registry semantics, output formatting algorithms, meta-command effects,
CLI argument handling, session lifecycle, and correctness guarantees.

== Conventions

- All specification files use the `@taxonomy: spec/*` annotation prefix
- Grammar changes must update both the original and this canonical copy
- SPEC.core.scm follows the RSR META-FORMAT-SPEC
